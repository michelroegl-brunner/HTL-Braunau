# BEISPIELDATEIN

In diesem Ordner liegen Beispiel-Dateien für Aufgaben aus Linux und PowerShell.

Struktur:

- `BEISPIELDATEIN/POWERSHELL/<AUFGABE>/...`
- `BEISPIELDATEIN/LINUX/<AUFGABE>/...`

Wichtige Input-Dateien (Log-Analyzer):

- `BEISPIELDATEIN/LINUX/Aufgabe-Logfile-Analyzer/app.log`
- `BEISPIELDATEIN/POWERSHELL/Aufgabe-Serie-3-Log-Analyse/app.log`
- `BEISPIELDATEIN/POWERSHELL/Aufgabe-Serie-3-CIM/rechnerliste.txt`

## Download von GitHub

Repo: [HTL-Braunau](https://github.com/michelroegl-brunner/HTL-Braunau)

### Einzelne Datei herunterladen

1. Datei auf GitHub öffnen.
2. Auf **Raw** klicken.
3. Rechtsklick -> **Speichern unter**.

Alternativ direkt mit Raw-Link:

[Raw-Link-Muster](https://raw.githubusercontent.com/michelroegl-brunner/HTL-Braunau/main/BEISPIELDATEIN/<AUFGABE>/<DATEI>)

### Ganzen Ordner herunterladen

1. Repo als ZIP herunterladen:  
   [Repo als ZIP herunterladen](https://github.com/michelroegl-brunner/HTL-Braunau/archive/refs/heads/main.zip)
2. ZIP entpacken.
3. Zum Ordner `BEISPIELDATEIN` wechseln.
